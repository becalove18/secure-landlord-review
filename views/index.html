RentReview protects user accounts and submitted reviews using secure web-development practices.

Secure Passwords

02

Protected Queries

Database queries use parameterized values to reduce SQL injection risk.

03

Input Validation

User input is checked and validated on the server before it is processed.

04

Authenticated Reviews

Only signed-in users are allowed to submit landlord reviews.

05

XSS Protection

User-generated content is escaped before being displayed on the website.
