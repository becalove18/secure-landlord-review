Secure Landlord Rating Website

This website allows renters to create accounts, log in, and share landlord and rental experiences.

Security Features

  • Passwords are hashed with bcrypt.
  • Database queries use parameterized values.
  • User input is validated on the server.
  • Only authenticated users can submit reviews.
  • User-generated output is escaped to reduce XSS risk.
